# Open Library 电子书刮削（替换 Bangumi）

## 目标

用 Open Library 替换 Bangumi 作为电子书元数据的自动刮削源。手动绑定端点保留 Bangumi 选项但自动刮削全部走 Open Library。搜不到的跳过。

## Open Library API 概要

- **搜索**: `GET https://openlibrary.org/search.json?q={query}&limit=5`
  - 返回 `docs[]`: `title`, `author_name[]`, `first_publish_year`, `key`(OLID), `cover_i`, `isbn[]`, `publisher[]`, `language[]`
- **封面**: `https://covers.openlibrary.org/b/olid/{olid}-L.jpg`（或用 ISBN）
- 无需 API Key，限速 1 req/s
- User-Agent 需标识应用

## 改动文件清单

### 1. 新建 `OpenLibraryService`
**文件**: `ebook/src/main/java/com/fryfrog/hub/ebook/service/OpenLibraryService.java`

```
@Service，注入 scraperRestTemplate + ObjectMapper

方法:
- searchBooks(String keyword) -> List<SearchResult>
  调用 /search.json?q={keyword}&limit=5
  返回搜索结果列表

内部类 SearchResult:
  - title (String)
  - authorName (String) — 取 author_name[0]
  - firstPublishYear (Integer)
  - olid (String) — 从 key 中提取，如 /works/OL12345W → OL12345W
  - coverId (Long)
  - isbns (List<String>)
  - publishers (List<String>)
  - language (String) — 取 language[0]
```

User-Agent: `FryfrogHub/0.1.0`（与现有 BangumiService 一致）

### 2. 修改 `EbookMetadataScrapeService`
**文件**: `ebook/src/main/java/com/fryfrog/hub/ebook/service/EbookMetadataScrapeService.java`

**改动点：**

- **注入 `OpenLibraryService`**（可移除对 `BangumiService` 的依赖，因为自动刮削不再需要）
  - **注意**：`searchBooks()` 和 `scrapeAndBind()` 方法仍在用 Bangumi 做手动搜索/绑定，不能移除 BangumiService
  - 只修改自动刮削流程

- **`autoScrapeEbook()` 重写**:
  ```
  1. query = extractSeriesName(ebook) 或 ebook.getTitle()
  2. openLibraryService.searchBooks(query)
  3. 有结果 → scrapeAndBindFromOpenLibrary(ebookId, results[0])
  4. 无结果 → 跳过，log.debug
  ```

- **新建 `scrapeAndBindFromOpenLibrary(Long ebookId, OpenLibraryService.SearchResult result)`**:
  ```
  - ebook.setTitle(result.title)
  - ebook.setAuthor(result.authorName)
  - ebook.setYear(result.firstPublishYear)
  - ebook.setIsbn(result.isbns[0])  // 如果有
  - ebook.setPublisher(result.publishers[0])  // 如果有
  - 封面: 下载到本地，文件名 ol_{olid}_cover.jpg
  - ebook.setOpenLibraryId(result.olid)  // 用这个字段标记来源
  - repository.save(ebook)
  ```

## 封面下载

复用现有 `downloadCover` 方法模式。Open Library 封面 URL:
- 优先用 OLID: `https://covers.openlibrary.org/b/olid/{olid}-L.jpg`
- 无 OLID 时用 ISBN: `https://covers.openlibrary.org/b/isbn/{isbn}-L.jpg`

## 自动刮削流程（改后）

```
autoScrapeAll():
  查所有 openLibraryId 和 bangumiId 都为空的 ebook（即从未绑定过的）
  逐个调用 autoScrapeEbook()

autoScrapeEbook(ebook):
  query = extractSeriesName(ebook) 或 ebook.getTitle()
  
  olResults = openLibraryService.searchBooks(query)
  if olResults 非空:
    scrapeAndBindFromOpenLibrary(ebookId, olResults[0])
    return
  
  // 搜不到，跳过
  log.debug("No Open Library results for '{}'", query)
```
